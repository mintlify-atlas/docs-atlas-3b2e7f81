# From zero to one

Dev0 is an agentic engineering platform that turns your ideas into working GitHub pull requests. You describe the feature, Dev0 plans it into tasks, runs each task inside an isolated sandbox using Gemini AI, and puts a reviewable PR in front of you — every time.

Engineers stay in control as technical leads. Dev0 handles the execution.

## The Dev0 loop

Dev0 is built around the Ralph loop — let a code agent run until the task is actually done, but keep it contained, auditable, and reviewable.

<Steps>
  <Step title="Describe your idea">
    Enter a rough "vibe" — a few sentences about what you want to build. Gemini 3 Flash turns it into a project name, tagline, description, and tech stack.
  </Step>
  <Step title="Review the plan">
    Gemini 3 Pro generates a dependency-aware task graph with phases, order, and model assignments. You see every task before execution starts.
  </Step>
  <Step title="Execute in isolation">
    Each task runs inside an E2B sandbox. Gemini CLI executes in YOLO mode — fully autonomous, fully contained. Live logs stream to your dashboard.
  </Step>
  <Step title="Review and merge">
    Every completed task produces a GitHub pull request. You approve, request changes, or merge directly from Dev0's mission control UI.
  </Step>
</Steps>

## Tech stack

Dev0 is built on modern, battle-tested technologies:

| Layer | Technology |
|---|---|
| Frontend | React 19, TanStack Start, TanStack Router, Vite |
| UI | shadcn/ui, Tailwind CSS v4, Motion, Lucide |
| Backend | TanStack Start server actions, TypeScript |
| Database | PostgreSQL, Drizzle ORM |
| AI | Gemini 3 Pro/Flash, Vercel AI SDK, Gemini CLI |
| Sandbox | E2B, Docker templates |
| Realtime | Upstash Realtime, Upstash Redis |
